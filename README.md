# Crim

Criminal Justice Education Website

## Adding Photos and Videos

To enhance the website with images and videos:

### Photos
- Place image files in the `images/` directory
- Supported formats: JPG, PNG, GIF
- Update the `src` attributes in the HTML files to match your image filenames
- Example: `<img src="images/arrest_flowchart.jpg" alt="Flowchart of the arrest process">`

### Videos
- For the documentary clip in `confession.html`, replace `YOUR_VIDEO_ID_HERE` with the actual YouTube video ID
- The iframe is set up for YouTube embeds
- If using a different platform, adjust the iframe src accordingly

### Image Suggestions
- arrest_flowchart.jpg: Visual flowchart of arrest process
- arrest_timeline.jpg: Timeline graphic
- probable_cause_diagram.jpg: Diagram comparing probable cause and reasonable suspicion
- search_chart.jpg: Chart for warrant vs. no warrant searches

- miranda_breakdown.jpg: Visual breakdown of Miranda rights
- interrogation_chart.jpg: Chart of legal vs. illegal techniques

## Running the Website
Open `index.html` in a web browser to view the site.
